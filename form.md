DARK UI STYLE GUIDE (COMPACT)
1. Tổng thể

Dark theme – Control Panel

Nền tối, panel chia khối rõ ràng

Viền & màu nhấn Vàng #FECD08

Font sans-serif, dễ đọc, không trang trí thừa

2. Text / Font

Màu chữ mặc định: #FFFFFF

Các kiểu chữ:

Bold (heading, label)

Regular (content)

Italic (note)

Không dùng nhiều font khác nhau

3. Color System
Text color

Primary accent: #FECD08

Default: #FFFFFF

Dark text (khi cần): #000000

Background

Main BG: #3A393C

Panel BG: #5D5D5F

Absolute dark / overlay: #000000

4. Input

Input nền tối, viền vàng

Text trắng, placeholder xám

Input số có nút + / -, giá trị ở giữa

Không cho nhập giá trị không hợp lệ

5. Button
Trạng thái bắt buộc

Normal

Hover

Disable

Màu button

Vàng: hành động chính

Xanh dương: thông tin / xác nhận

Xanh lá: thành công

Đỏ: nguy hiểm / xoá

Disable: xám, mờ, không click

6. Text Link

Normal: vàng

Hover: sáng hơn / underline

Disable: xám, không tương tác

Không có nền

7. Icon

Style: flat, minimal

Màu:

Default: trắng

Hover / active: vàng

Disable: xám

Dùng icon phổ biến (setting, user, item, money…)

8. Layout

Panel có padding rõ

Khoảng cách đều giữa các khối

Căn trái nội dung

Không dồn sát viền

9. Quy tắc cho AI / Dev

Không tự thêm màu mới

Không đổi layout

Luôn đủ state UI

Ưu tiên phong cách Game UI / FiveM gtav / Control Panel
